A specialized Secondo that adapts external event sources â€” either a DOM EventTarget or a Node.js-style EventEmitter â€” into the Ensemble system.

Responsibilities:

  • Attaches to an external emitter and listens for configured events.
  • Re-emits incoming events to the Ensemble hierarchy.
  • Appends itself as the last argument to propagated events, allowing listeners to access both the EmitterSecondo and the underlying emitter.

Example usage:

const button = document.querySelector('button');
const buttonSecondo = new EmitterSecondo(button, ['click']);

buttonSecondo.on('click', (event, secondo) => {
  console.log('Clicked element:', secondo.primo);
});

buttonSecondo.play();
