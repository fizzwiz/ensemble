A Player that performs a repeating action indefinitely (or a fixed number of times).

Musically, an ostinato is a short phrase or rhythm that is persistently repeated. In this class, the repeated task is represented by a periodic AsyncWhat that can be started or stopped via play() and pause().

Complements Finale, which represents a finite-duration performance.
