Secondo

A Solo that wraps a non-Player event source and forwards its events alike emissions, making it compatible with the Ensemble system.

This enables integrating third-party or native event emitters (e.g., WebSocket, DOM elements, etc.) into the Ensemble-based cueing and propagation framework.
